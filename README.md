# 🚀 Prompt Inteligente

Uma aplicação web que utiliza Inteligência Artificial para gerar landing pages completas a partir de um simples tema informado pelo usuário.

O projeto integra HTML, CSS, JavaScript e a API da GroqCloud para transformar uma descrição em uma página web pronta, com design moderno, estrutura organizada e conteúdo personalizado.

# 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/Denilson-Ferreira/Prompt-inteligente.git
```

Entre na pasta:

```bash
cd Prompt-inteligente
```

Abra o arquivo:

```
index.html
```

ou utilize a extensão **Live Server** do VS Code.

---

# 🔑 Configuração da API

Por segurança, nenhuma chave de API é armazenada neste repositório.

Crie uma variável contendo sua chave da GroqCloud:

```javascript
const API_KEY = "SUA_CHAVE_AQUI";
```

> **Importante:** Nunca publique sua chave da API em um repositório público.

---

# 💡 Como funciona

1. O usuário informa um tema para o site.
2. O JavaScript envia a solicitação para a API da GroqCloud.
3. A IA gera uma landing page completa.
4. O HTML retornado é exibido na tela.
5. O usuário pode visualizar o código e a página renderizada.
